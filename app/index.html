Typography

Base Type

For our base we use the following font settings on our body element to standardize the typographic scale for us in a consistent manner without the dependency of a library or reset style sheet.

$weight: normal;
$lineheight: 1.5;
$fontsize: 100%;
$font-family: sans-serif;
$font-properties: $weight, $lineheight, $fontsize, $font-family;

@mixin body-copy($weight, $lineheight, $fontsize, $font-family...) {
   font: $weight #{$fontsize}/#{$lineheight} $font-family;
}

body {
    @include body-copy($font-properties...);
}

Font Size Adjust

Using font-size-adjust on our body element we help to maintain a consistent x–height across our fallback fonts. This is currently not implemented across browsers, but due to become a standard very soon. In essence, you're preparing to become future friendly with your type. It was pulled from CSS2.1, but brought back to life in the CSS Fonts Module Level 3.

x

x

body {
    font-size-adjust: 1;
}

$heading3: 18;

@function calc-modular-scale($scale, $base, $value) {
    @return ($scale/$base)#{$value};
}

@mixin modular-scale($scale, $base, $value) {
  font-size: calc-modular-scale($scale, $base, $value);
}

h1 {
    @include modular-scale($heading1, $fontbase, 'rem'); //2.25rem
}

Color

Just like in print, don’t use the darkest black available in your body text (#000). Instead, go for something softer, like #444. You can use a slightly darker color for titles (#222).

//color
body { color: #444; }
h1,h2,h3,h4,h5,h6 { color: #222; }

Unicode Ampersands

With the unicode–range property we can avoid using verbose markup and only target the charcters we like according to the unicode unique identifier.

Amper & Sands

//Unicode Ampersand
$amp-font: local('Baskerville'), local('Palatino'), local('Book Antiqua');
@font-face {
    font-family: 'Ampersand';
    src: $amp-font;
    unicode-range: U+26;
}
//fallback for unicode range
@font-face {
    /* Ampersand fallback font */
    font-family: 'Ampersand';
    src: local('Arial');
    unicode-range: U+270C;
}

* {
    font-family: Ampersand, Arial, sans-serif;
}
